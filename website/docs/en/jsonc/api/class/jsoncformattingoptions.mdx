# JsoncFormattingOptions

Options controlling JSONC formatting. All fields are omissible; the formatter resolves absent fields to defaults: `tabSize` 2, `insertSpaces` `true`, `eol` `"\n"`, `insertFinalNewline` `false`, `keepLines` `false`.
