# JsoncParseOptions

Options controlling parse behavior. All fields are omissible; absent fields resolve to `disallowComments` `false`, `allowTrailingComma` `true` (the deliberate JSONC-convention default, differing from Microsoft's parser) and `allowEmptyContent` `false`.
