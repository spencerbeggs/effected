# VersionCacheShape

Operations of the VersionCache service. Every query is a thunk; queries over the whole cache (`versions`, `filter`) never fail and return `[]` when nothing matches, while extremum and navigation operations fail typed. `next`/`prev` layer two different absences deliberately: the error channel means "the pivot version is not cached", `Option.none()` means "the pivot is at the boundary".
