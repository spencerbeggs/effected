# VersionDiff

The difference between two SemVer versions: the classification of the change plus signed numeric deltas. A `Schema.TaggedClass` — the one concept in this package where serialized tag discrimination earns its keep. The `type` field is the highest-precedence field that differs: `"major"`, `"minor"`, `"patch"`, `"prerelease"` (only prerelease identifiers differ), `"build"` (only build metadata differs) or `"none"`.
